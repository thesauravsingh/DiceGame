# DiceGame
This is a simple web-based dice game built using HTML, CSS, and JavaScript. It's designed for two players to roll dice and compete against each other to reach a winning score.

How to Play
Open index.html in your web browser.
Click on the "Roll Dice" button to roll the dice for Player 1.
Player 1 and 2's score will be displayed.
The player with the highest score wins the round.
Continue playing rounds until one player reaches the winning score.

Features
Interactive interface.
Dynamic display of scores and winner.
Easy to understand and play.
